Thuộc tính style trong thẻ HTML
        style=""
        Bên trong nháy kép sẽ là thuộc tính của thẻ CSS
        CSS là một thẻ trang trí website
        code trong thẻ style=""
        nó sẽ đi theo cặp"tên_thuộc_tính:giá_trị"
        Trường hợp có nhiều thuộc tính css trong thuộc tính html style
        thì cú pháp như sau"tên_thuộc_tính 1:giá_trị;tên_thuộc_tính_2:giá trị;...
        Làm quen với một số thuộc tính cơ bản của css:
        color(màu sắc trong css)
        background-color(màu sắc của nền)
        font-size(kích cỡ chữ trong css)
