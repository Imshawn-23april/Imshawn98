Thuộc tính id trong html và ứng dụng của nó
        Vấn đề như sau : có 5 thẻ p chứa 5 tỉnh thành khác nhau
        thẻ p chứa ha noi ( màu đỏ )
        thẻ p chứa ho chi minh ( màu vàng )
        thẻ p chứa da nang ( màu xanh )
        thẻ p chứa can tho ( màu hồng )
        thẻ p chứa hai phong ( màu da cam )
        Nếu sử dụng selector ( bộ chọn ) theo tên thẻ thì không làm được
        Nếu sử dụng dạng Inline style thì sẽ làm được
        Nhưng nếu sử dụng Internal style thì không thể sử dụng selector là p được
        do cả 5 thẻ đều là thẻ p
        Kết luận: trong html cung cấp 1 thuộc tính là id
        thuộc tính này có tác dụng định danh cho 1 thẻ html duy nhất
        Giá trị của id phải viết liền và không dấu dồng thời ngắn gọn nhất có thể
        Chú ý: khi html có thuộc tính thì viết code css sẽ sử dụng selector có id tương ứng
        => #id{
                code css
        }
